# Discord Bot Panel

Simple website displaying information about Discord bot:
  * guilds count
  * members count
  * executed commands count

## Technologies
**Backend**
  * ASP.NET Core
  * Entity Framework Core with SQLite provider
  * NLog
  * AutoMapper
  * xUnit

**Frontend**
  * Angular 6
  * Redux
  * Bootstrap
  * Chartjs
